Hamburgueria : Planilha de Requisitos e Regras de Negócio

Para este projeto, desenvolvi a documentação completa de uma hamburgueria, organizando todos os elementos essenciais para o seu planejamento e operação. Criei um documento detalhado contendo:

Identidade visual: Inseri o logo e o nome da hamburgueria, destacando sua proposta de valor (como conceito e público-alvo).

Requisitos funcionais: Liste as funcionalidades necessárias para o negócio, como sistema de pedidos, cadastro de clientes e controle de estoque.

Requisitos não-funcionais: Defini critérios de performance, segurança e usabilidade para garantir uma experiência eficiente.

Regras de negócio: Estabeleci normas operacionais, como políticas de promoções, horários de funcionamento e condições de personalização dos lanches.

Utilizei uma planilha para estruturar as informações de forma clara e acessível, facilitando a visualização e o alinhamento com a equipe. Essa atividade reforçou minha capacidade de organizar requisitos e regras de negócio, essenciais para qualquer projeto bem-sucedido."

Habilidades trabalhadas: H4, H7, H8

S.A - Apresentação 01 (Conhecendo a marca e os requisitos do sistema)

Desenvolvi, em equipe, o projeto SmartSitter – um dispositivo inteligente de monitoramento infantil que permite o acompanhamento remoto de bebês através de vídeo, áudio e controle de temperatura ambiente. O trabalho envolveu a criação de um conceito inovador, desde a etapa inicial de ideação até o desenvolvimento de protótipos e requisitos técnicos.

Utilizamos ferramentas como Figma para elaborar wireframes do aplicativo e site, garantindo uma interface intuitiva para os usuários. O projeto incluiu a definição de identidade visual, com paleta de cores e logomarca, além da especificação de funcionalidades como transmissão em tempo real, alertas personalizáveis e sistemas de segurança para proteção de dados.

Habilidades trabalhadas: H1, H2, H3, H4, H6, H7
